# Zwiggato

Zwiggato is a React-based food delivery web app built using Parcel.

## Features

- React application structure with reusable components
- Client-side routing with `react-router-dom`
- Responsive UI with custom styles
- Simple entry point for development using Parcel

## Project Structure

- `index.html` - Application shell and root container
- `styles.css` - Global styling for the app
- `src/app.js` - React app entry point
- `src/components/` - Reusable UI components
  - `Header.js`
  - `Body.js`
  - `RestaurantCard.js`
  - `Shimmer.js`
  - `About.js`
  - `Contact.js`
  - `Error.js`
- `src/utils/constants.js` - Shared constants used by the app

## Getting Started

### Prerequisites

- Node.js
- npm

### Install dependencies

```bash
npm install
```

### Run the app

```bash
npm start
```

Then open the local development server shown in the terminal.

## Scripts

- `npm start` - Starts the Parcel development server
- `npm test` - Runs the test runner (if tests are added)
